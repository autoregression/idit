# IDiT - Iterative Diffusion Transformer

A DiT architecture that leverages iteration to squeeze out more performance per layer.

Installation 
------------

```
git clone https://github.com/autoregression/idit

uv sync
```

Usage
-----

Training:

```
PUSH_PATH='username/repo' uv run train.py 
```

Sampling:

```
PUSH_PATH='username/repo' uv run sample.py
```
